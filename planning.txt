Voting App

For starting any backend project first plan for 2 things
1. Model Schema
2. Route or endpoints

What is the functionality??

1. User SignUp
2. Candidate list shown to users
3. Only one vote per person
4. Realtime vote Count
5. Aadhar is added for uniqueness
6. Admin page for adding or updating the info. of the candidates
7. Password can be changed by user using aadhar
8. Aadhar and password for login
9. Admin can't vote

Routes

1. USER AUTH:- post("/signup"):- to create user
               post("/login"):- to login to account
2. VOTING PHASE:- get("/candidate):- list of candidates
                  post("/vote/:candidateID"):- to vote to specific person
3. VOTE COUNT:- get("/vote/count"):- show realtime votes
4. USER PROFILE:- get("/profile:):- user profile
                  put("/profile"):- update password
5. ADMIN CONTROL:- post("/candidate"):- to add candidate
                   put("/candidate/candidateID"):- to update candidate info.
                   delete("/candidate/candidateID"):- to delete candidate info.
